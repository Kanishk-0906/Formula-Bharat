# Formula Bharat – Formula SAE Race Car Development

This repository contains my contributions to the Formula Bharat Formula SAE race car project at IIT Mandi.

## Overview

As a member of the Formula Bharat team, I contributed to the design, simulation, and validation of critical vehicle subsystems, including steering and powertrain components.

## Contributions

### Steering System Design
- Designed upper and lower steering assemblies in SolidWorks.
- Optimized rack parameters using MATLAB.
- Validated stress distribution and deflection through ANSYS FEA prior to fabrication.

### Chassis Design
- Contributed to the development of a Formula Bharat spaceframe chassis.
- Incorporated triangulated load paths, crumple zones, and DFM principles.

### Powertrain Engineering
- Assisted in the conversion from an electric to IC powertrain.
- Recalibrated ECU parameters to improve torque delivery and fuel efficiency.

## Tools Used

- SolidWorks
- ANSYS Static Structural
- MATLAB
